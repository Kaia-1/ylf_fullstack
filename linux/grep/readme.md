- gitbash 是个linux 最小化战场
- grep 命令 用于查找
      grep moon monkey
    -v 查找反向条件      <!--grep -v moon monkey -->
    -c 符合条件的数量    <!--grep -c moon monkey -->
    -i 大小写忽略        <!--grep -i my monkey -->
    -n 行号输出         <!--grep -n monkey monkey -->
    ^  开始             <!--grep '^J' monkey -->
    $ 结束              <!--grep '呢$' monkey -->
